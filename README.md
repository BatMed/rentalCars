# Project Starter

## Setup

Make sure to install the dependencies:

```bash
# npm
npm install

# pnpm
pnpm install

# yarn
yarn install

# bun
bun install
```
## CORS Anywhere
Employ "CORS Anywhere proxy" to access CommonShare API from "localhost"
```
https://cors-anywhere.herokuapp.com/corsdemo
```
## Development Server

Start the development server on `http://localhost:3000`:

```bash
# npm
npm run dev

# pnpm
pnpm run dev

# yarn
yarn dev

# bun
bun run dev
```
